# 🕰️ TimeTravel Agency — Voyagez à travers le temps
# Groupe Evan BOYER, Thomas ZABALO, Elias NODON


**TimeTravel Agency** est une application web moderne et immersive permettant de simuler l'expérience d'une agence de voyage temporelle. Ce projet met en avant des designs futuristes, des animations fluides et une interface utilisateur "production-ready" construite avec React.

---

## 🚀 Technologies Utilisées

Ce projet repose sur une pile technologique moderne pour garantir performance et flexibilité :

- **Frontend** : [React 18](https://reactjs.org/) (Hooks, Architecture Composants)
- **Build Tool** : [Vite 5](https://vitejs.dev/) (Développement ultra-rapide)
- **Styling** : [Tailwind CSS](https://tailwindcss.com/) & [CSS3 Variables](https://developer.mozilla.org/en-US/docs/Web/CSS/Using_CSS_custom_properties) (Thémage et mise en page responsive)
- **Animations** : [Framer Motion](https://www.framer.com/motion/) (Transitions et effets d'entrée sophistiqués)
- **Typographie** : Combinaison de polices serif élégantes pour les titres et sans-serif pour la lisibilité.

---

## ✨ Features Implémentées

- ⚡ **Performance Optimisée** : Chargement rapide et transitions fluides.
- 📱 **Responsive Design** : Expérience utilisateur adaptée du mobile au desktop.
- 🧭 **Navigation Intuitive (Navbar)** : Menu collant avec effets de survol.
- 🎭 **Hero Section Immersive** : Entrée animée décalée et visuels flottants.
- 🗺️ **Destinations Temporelles** : Exploration de lieux comme la Rome Antique ou le Tokyo futuriste.
- ⏳ **Timeline Historique** : Visualisation chronologique des événements de voyage.
- 💳 **Pricing Dynamique** : Présentation des forfaits (Bronze, Silver, Gold).
- 🤖 **Chatbot IA** : Assistant virtuel intégré pour répondre aux voyageurs.
- 🖼️ **Effets de Cartes Inclinées (Tilted Cards)** : Interactivité visuelle avancée.

## ⚙️ Instructions d’Installation

Pour lancer le projet localement sur votre machine :

### Prérequis
- [Node.js](https://nodejs.org/) (Version 16 ou supérieure recommandée)
- npm ou yarn

### Étapes

1. **Cloner le repository** :
   ```bash
   git clone <url-du-repo>
   cd TimeTravelAgency
   ```

2. **Installer les dépendances** :
   ```bash
   npm install
   ```

3. **Lancer le serveur de développement** :
   ```bash
   npm run dev
   ```

4. **Accéder à l'application** :
   Ouvrez votre navigateur sur `http://localhost:5173`.
